# 3---Desafio-Mentoria-
Este desafio de mentoria foca no aprendizado de Tailwind CSS e na autonomia do desenvolvedor. O participante recriará a interface Festivite e fará uma Landing Page responsiva com animações, usando apenas HTML e Tailwind . É proibido usar IA, e toda pesquisa deve ser em inglês. O processo exige planejar o projeto via Issues no GitHub.
