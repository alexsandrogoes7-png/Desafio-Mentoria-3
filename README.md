# Desafio Mentoria #3: Aprendendo Tailwind CSS e Habilidades Básicas

### 🎯 Desafios Práticos

1. **Desafio 01:** Recriar a interface do *Festivite* (Formulário de Convite) utilizando Tailwind CSS.
2. **Desafio 02:** Desenvolver a Landing Page (LP) de um Clube de Assinatura, focando na aplicação de animações com Tailwind.
3. **Desafio 03:** Finalizar os detalhes e interações avançadas do desafio da LP do Clube de Assinatura.

> **Observação:** Caso o Desafio 02 se mostre muito complexo no início, você tem a liberdade de inverter a ordem de execução com o Desafio 03. O mais importante é não deixar de tentar!

### 🧠 Quiz Técnico

Antes ou durante o desenvolvimento, você deve ser capaz de responder:

* Como configurar e habilitar o Tailwind CSS em projetos com HTML tradicional (Vanilla)?
* Como implementar a alternância de temas (Dark/Light Mode) utilizando Tailwind CSS e HTML tradicional?

### ⚠️ Regras de Execução

* **Responsividade:** O layout deve se adaptar perfeitamente a dispositivos móveis, tablets e desktops (*Mobile First*).
* **Stack Permitida:** Utilizar apenas HTML tradicional, Tailwind CSS e JavaScript (apenas se houver necessidade de interatividade no DOM).
* **Zero IA:** É terminantemente proibido o uso de Inteligências Artificiais (ChatGPT, Claude, Copilot, etc.) para gerar código ou tirar dúvidas.
* **Pesquisa Restrita:** Todas as buscas por soluções e dúvidas devem ser feitas **exclusivamente em inglês**.
* **Fontes de Estudo:** Consulte apenas documentações oficiais, blogs, artigos ou tutoriais no idioma inglês.
* **Fidelidade ao Design:** O resultado final deve ser o mais fiel possível ao protótipo do Figma, incluindo todas as transições e animações propostas.

### 🚀 Propósito da Mentoria

* Aprimorar suas habilidades técnicas como programador.
* Desenvolver o inglês instrumental focado em tecnologia.
* Praticar a leitura e a interpretação de documentações oficiais.
* Estimular a autonomia e a tomada de decisão na resolução de problemas estruturais.
